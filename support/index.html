Common Issues

The extension button isn't appearing on a form

Some websites use JavaScript to render forms dynamically. Try scrolling down to trigger the page to fully load, then look for the ⚡ button. If it still doesn't appear, email us with the URL of the site.

Gemini Nano says "Not supported on this device"

Gemini Nano requires Chrome 127+ and sufficient device RAM (typically 8GB+). You can still use the extension by adding your own API key in Settings (Bring Your Own Key). OpenAI, Anthropic, Groq, and OpenRouter are all supported.

My profile isn't saving

Go to the extension's Options page (right-click extension icon → Options). Make sure you've clicked Save after entering your profile. If it still doesn't save, try the Troubleshoot button on the Options page.

The AI-generated answer doesn't sound like me

The AI uses your profile to generate answers. Fill in the Bio and Experience fields with detail — the more context you give it, the better the output. You can always edit any answer before submitting.

How do I uninstall and remove my data?

In Chrome, go to chrome://extensions, find Donbrico Autofill AI, click Remove. All your profile data will be deleted automatically.

How do I report a bug or request a feature?

Email support@donbrico.net with "Bug:" or "Feature Request:" at the start of your subject line.

Null Carrier Support

For issues with Null Carrier on Steam, you can reach us through Steam's support system or directly at support@donbrico.net.

Please include your Steam username and a description of the issue.

Response time policy

Donbrico is a solo development operation. Support is provided on a best-effort basis. We aim to respond within 1–2 business days. Critical extension bugs are prioritized.
