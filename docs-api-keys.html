Donbrico Autofill AI works with four AI providers. Pick the one that suits you — each takes under 5 minutes to set up and charges only for what you actually use. Groq and OpenRouter even have completely free tiers with no credit card required.

⚡

Provider 01

⚡ Groq

console.groq.com
Free tier available Llama 3 · Mixtral · Gemma
Create account ↗

Groq runs open-source models on custom silicon (LPUs) — the result is remarkably fast inference. The free tier is generous enough for everyday autofill use at no cost.

1

Sign up for a free Groq account

Go to console.groq.com and click Sign up. Groq supports Google, GitHub, and email signup. No credit card is required for the free tier.

2

Create an API key

In the left sidebar, click API Keys. Click Create API Key, enter a name like "Donbrico", and click Submit.

⚠️ Copy the key immediately. Groq does not show it again after creation. It begins with gsk_…
3

Paste into the extension

Open the extension popup → Settings → AI Provider → Groq. Paste your key and save.

ℹ️
Groq free tier includes rate limits (e.g. 30 requests/minute for Llama 3). For typical autofill usage this is more than enough. If you hit limits, upgrading to a paid plan removes them.

Provider 02

🔀 OpenRouter

openrouter.ai
Free models available 100+ models from any provider
Create account ↗

OpenRouter is a unified gateway to over 100 AI models — including GPT-4o, Claude, Llama, Mistral, and many more — through a single API key. Several models are permanently free with no credit card required.

1

Create an OpenRouter account

Visit openrouter.ai and click Sign In. OpenRouter supports Google and GitHub sign-in, or you can use an email address.

2

Optional: add credits for paid models

Go to Credits in your account and top up if you want to use paid models like GPT-4o or Claude. For free models only, no payment is needed.

💡 Filter by Free on the Models page to see the full list of zero-cost models available right now.
3

Create an API key

Go to Keys in the top navigation. Click Create Key, name it "Donbrico", and copy the key — it begins with sk-or-…

⚠️ Copy the key now — it will be hidden after you close this dialog.
4

Paste into the extension

Open the extension popup → Settings → AI Provider → OpenRouter. Paste your key, choose your preferred model, and save.

Provider 03

🟣 Anthropic

console.anthropic.com
Pay-as-you-go Claude 3.5 Haiku · Claude 3.5 Sonnet
Create account ↗

Anthropic's Claude models are known for following instructions precisely — ideal for generating well-structured cover letters and professional answers.

1

Create an Anthropic account

Visit console.anthropic.com and click Sign up. You can use Google or an email address.

2

Add billing information

From the Console dashboard, go to Settings → Billing. Click Add credit card and complete the form. Anthropic is pay-as-you-go with no monthly fee.

💡 You can set a monthly spend limit under Billing → Usage limits to avoid any surprise charges.
3

Generate an API key

In the left sidebar, click API Keys. Click Create Key, name it "Donbrico", and copy the key — it starts with sk-ant-…

⚠️ The key is shown only once at creation time. Copy it before closing the dialog.
4

Paste into the extension

Open the extension popup → Settings → AI Provider → Anthropic. Paste your key and save.

Provider 04

🟢 OpenAI

platform.openai.com
Pay-as-you-go GPT-4o · GPT-4o mini
Create account ↗

OpenAI is the most widely used provider. GPT-4o mini is fast, accurate, and costs fractions of a cent per autofill.

1

Create a free account

Go to platform.openai.com and sign up. You can use Google, Microsoft, or email.

2

Add a payment method

Navigate to Settings → Billing → Add payment method. OpenAI requires a credit card before generating API keys. You will only be charged for actual usage — new accounts receive $5 in free credits.

3

Create an API key

Go to API Keys in the left sidebar (or visit platform.openai.com/api-keys). Click Create new secret key, give it a name like "Donbrico", and click Create secret key.

⚠️ Copy your key immediately — OpenAI only shows it once. Store it somewhere safe.
4

Paste the key into the extension

Open the Donbrico Autofill AI popup → Settings → AI Provider → OpenAI. Paste your key (it starts with sk-…) and click Save.
